# 🚧 Pothole Detection using YOLO

## 📌 Project Overview

This project focuses on detecting potholes on roads using **YOLO (You Only Look Once)**, a state-of-the-art real-time object detection algorithm.
The system processes images/videos and identifies potholes to improve road safety and maintenance efficiency.

---

## 🎯 Objectives

* Detect potholes in real-time using deep learning
* Improve road safety by early detection
* Assist smart city infrastructure and maintenance systems

---

## 🧠 Technologies Used

* Python 🐍
* OpenCV
* YOLO (v5 / v8)
* NumPy

---

## ⚙️ Features

* Real-time pothole detection
* Works on images and videos
* High accuracy using trained YOLO model
* Easy to run and modify

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository

```
git clone https://github.com/dhakarvinay/Pothole-Detection-using-YOLO.git
cd Pothole-Detection-using-YOLO
```

### 2️⃣ Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run Detection

```
python predict.py
```
